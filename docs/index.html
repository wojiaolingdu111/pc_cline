AI ToReder

基于 Tauri + Rust + Python 的桌面端本地 TTS 工作台。
语音合成、声音克隆，一切都在本地运行。

↓ 下载安装包 GitHub 仓库

功能特性

开箱即用的本地语音能力

🎙️

TTS 语音合成

基于 Coqui XTTS v2 模型，支持中文、英文等多语言合成，可调节语速，生成自然流畅的语音。

🧬

声音克隆

上传 6 秒以上的参考音频即可克隆声音，用你的音色朗读任意文本，支持多音色管理。

🖥️

跨平台桌面应用

基于 Tauri 2 构建，原生性能、低内存占用。支持 Windows、macOS、Linux 三大平台。

🔒

纯本地运行

所有计算在本地完成，无需联网，不需要任何 API Key，你的数据不会离开你的电脑。

📦

一键安装

Python 引擎 + 模型推理全部打包为独立 sidecar，安装后开箱即用，无需额外配置环境。

🎯

任务管理

内置任务队列和音频播放器，支持批量生成、导出 WAV 音频文件，方便后续使用。
